******************************例程详细描述*****************************
功能描述:
本例程描述了GPIO外部中断触发功能。

配置PA1为输入模式，PA2为输出模式，配置PA1上升沿触发中断，连接EXTI_Line1事件线。
当程序开始运行，检测到PA1上有上升沿时触发中断，中断内将PA2拉高。

时钟配置:
本例程采用HSI56MHz作为系统时钟，关于系统时钟如何配置，请参考对应
用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA系列

涉及外设:
GPIO
涉及管脚:
PA1, PA2

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估板，重启评估板;
2. 给PA1引脚输入一个上升沿；
3. 通过示波器或万用表观察PA2引脚输出状态;
