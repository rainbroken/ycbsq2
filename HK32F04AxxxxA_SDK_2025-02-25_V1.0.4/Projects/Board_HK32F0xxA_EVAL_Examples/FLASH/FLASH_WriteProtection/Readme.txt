******************************例程详细描述*****************************
功能描述:
本示例介绍了启用和禁用写保护的操作步骤用于HK32F0xxA嵌入式FLASH。

―启用写保护:
要启用写保护，开启宏定义“#define WRITE_PROTECTION_ENABLE”。
在main.c文件中。为了保护一组页面，用户必须调用FLASH_EnableWriteProtection函数。
该函数的参数将定义要保护的页面数量。要加载新的选项字节值，
必须进行系统重置使用FLASH_OBLaunch()函数。

―取消写保护:
要禁用写保护，开启宏定义“#define WRITE_PROTECTION_DISABLE”。
在main.c文件中。要禁用HK32F0xxA Flash的写保护，需要擦除选项字节是必要的。
该操作由FLASH_EraseOptionBytes函数完成。
要加载新的选项字节值，必须进行系统重置使用FLASH_OBLaunch()函数。

如果需要的页面不受写保护，则需要进行擦除和写操作执行。

时钟配置:
本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA

涉及外设:
无

涉及管脚:
无
