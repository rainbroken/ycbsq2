******************************例程详细描述*****************************
功能描述:
此示例显示如何定期更新IWDG重新加载计数器和如何模拟软件故障，
使用IWDG窗口模式，在到期时生成MCU IWDG重置.

IWDG重新加载和窗口值分别设置为2000和1000。
在这种情况下，刷新窗口为：（2000 - 1000）
1000 / （LSI / 16） = ~ 400 ms < 刷新窗口 < 2000 / （LSI / 16） = ~ 800 ms
@注意，刷新窗口可能因LSI频率分散而变化

IWDG计数器在主程序无限循环中每750毫秒刷新一次防止IWDG重置并伴随LED2闪烁。
EXTI Line用于模拟软件故障：一旦发生EXTI Line事件
发生时，按下Tamper（PC13）按钮，将提供相应的中断且LED3点亮。
此时操作一个非法地址造成错误进而无法喂狗，MUC重置
此时LED1点亮，串口1输出调试信息
如果EXTI Line事件未发生，IWDG计数器将无限期刷新
在主程序无限循环中，并且没有IWDG重置。

时钟配置:
本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA

涉及外设:
IWDG

USART1
USART1配置如下:
―波特率 = 115200
―字长 = USART_WordLength_8b
―停止位 = USART_StopBits_1
―奇偶校验 = USART_Parity_No
―硬件流控制被禁用(RTS和CTS信号)
―使能接收和发送

涉及管脚:
USART1 TX - PA9
USART1 RX - PA10

BUTTON_TAMPER - PC13

LED1 - PC10
LED2 - PC11
LED3 - PC12

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。
如何使用:
1. 直接编译程序下载到评估板，重启评估板;
