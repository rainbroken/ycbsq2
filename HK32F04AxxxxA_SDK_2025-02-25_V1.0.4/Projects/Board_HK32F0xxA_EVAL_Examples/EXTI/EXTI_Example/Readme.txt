******************************例程详细描述*****************************
功能描述:
本例程共配置四个EXTI线（EXTI0, EXTI8, EXTI9, EXTI13），EXTI0配置为上升沿触发中断，
EXTI8, EXTI9, EXTI13配置为下降沿中断，在对应中断程序中，翻转连接到LED的GPIO。

本例程中, PA0被连接到EXTI0, PC8被连接到EXTI8, PC9被连接到EXT9, PC13被连接到EXT13。
时钟配置:
本例程采用HSI56MHz作为系统时钟，关于系统时钟如何配置，请参考对应
用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA系列

涉及外设:
GPIO
EXTI
涉及管脚:
EXTI:
PA0, PC8, PC9, PC13
LED1:
PC10
LED2:
PC11
LED3:
PC12
LED4:
PD2

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估板，重启评估板;
2. 给EXTI线对应GPIO一个上升沿或下降沿；
3. 观察LED灯是否被点亮 / 熄灭。
