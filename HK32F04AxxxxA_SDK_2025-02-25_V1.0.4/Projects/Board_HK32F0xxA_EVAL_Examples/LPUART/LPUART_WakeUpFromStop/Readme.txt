******************************例程详细描述*****************************
功能描述:
本示例介绍了如何使用LPUART的“从STOP唤醒”模式功能

首先，本次例程LPUARTx使用的LPUART1，配置其为：
STOP唤醒方式为StartBitMethod（当在RX线上接收到任何数据时）。
基本配置完成后，MCU进入STOP模式，等待唤醒。
当字符串通过PC端串口助手发送到LPUART1时，MCU从停止唤醒，
恢复系统时钟配置并将系统时钟配置为切换LED。
四个LED的切换用于指示MCU从停止状态唤醒。

时钟配置:
本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA

涉及外设:
LPUART1
涉及管脚:
LPUART1 TX - PB6
LPUART1 RX - PB7

LED1 - PC10
LED2 - PC11
LED3 - PC12
LED4 - PD2

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。
如何使用:
1. 直接编译程序下载到评估版，重启评估版;
