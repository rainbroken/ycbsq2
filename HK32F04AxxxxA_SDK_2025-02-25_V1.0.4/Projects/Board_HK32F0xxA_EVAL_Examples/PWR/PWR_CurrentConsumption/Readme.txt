******************************例程详细描述*****************************
功能描述:
本例程描述了各种低功耗模式下的MCU工作电流，可配置以下低功耗模式：
-睡眠模式（外部中断唤醒）
-停止模式（RTC唤醒）
-停机模式（Wakeup Pin 唤醒）
-停机模式（RTC唤醒）

需要通过宏定义选择对应低功耗模式：
#define SLEEP_MODE
#define STOP_MODE
#define STANDBY_MODE
#define STANDBY_RTC_MODE

通过按下按键(PC13)进入低功耗模式，在睡眠模式和通过Wakeup Pin唤醒的停机模式下，
通过按下按键（PA0）从低功耗模式唤醒。在通过RTC唤醒的停止模式下，进入低功耗模式
后，RTC计时五秒后通过RTC闹钟唤醒。在通过RTC唤醒的停机模式下，进入低功耗后，RTC
计时八秒后通过RTC闹钟唤醒。
从睡眠模式或停止模式唤醒后，LED4(PD2)闪烁，表示唤醒成功。
时钟配置:
本例程采用HSI56MHz作为系统时钟，关于系统时钟如何配置，请参考对应用户手册或者
RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA系列

涉及外设:
PWR
RTC
涉及管脚:
KEY:
PA0, PC13
LED4:
PD2

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估板，重启评估板;
2. 通过按下按键（PC13）进入低功耗模式;
