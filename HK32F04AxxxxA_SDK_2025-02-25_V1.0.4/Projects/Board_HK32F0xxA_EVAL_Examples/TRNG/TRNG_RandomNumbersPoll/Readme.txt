******************************例程详细描述*****************************
功能描述:
这个示例描述了如何配置TRNG生成32位长随机数。

reset启动和系统配置完成后，进行TRNG配置。

然后要求用户按下按键(LED快速闪烁)。用户按下按钮，
生成几个(8)随机32位数字(DRDY标志被轮询到1，
表示已经生成了一个随机数，并且可以从DR寄存器中检索)。
相应的生成值可用并存储在u32数组(aRandom32bit)中，
其内容可以使用调试器(Watch或LiveWatch功能)显示, 并且通过串口打印出来。
随机数生成成功后，LED打开。如果出现错误，LED缓慢闪烁。

时钟配置:
本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA

涉及外设:
USART1
GPIO
涉及管脚:
USART1 PA9 TX
PA10 RX
KEY       PB8
LED       PC11

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。
如何使用:
1. 直接编译程序下载到评估版，重启评估版;
