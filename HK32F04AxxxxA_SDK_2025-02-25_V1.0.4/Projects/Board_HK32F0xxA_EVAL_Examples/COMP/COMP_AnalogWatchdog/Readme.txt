******************************例程详细描述*****************************
功能描述:
本例程描述用COMP配置模拟看门狗功能。

通过配置COMP1和COMP2设置窗口门限如下：
窗口上限阈值：VREFINT = 1.2V
窗口下限阈值：VREFINT / 4 = 0.3V

模拟输入电压通过PA1引脚输入，当模拟电压在窗口门限内时，系统进入停止模式，
如果在停止模式下，当输入电压在门限外时，将从停止模式唤醒芯片。
电压低于下限阈值时，LED2和LED4被点亮，电压高于上限阈值时，LED1和LED3被点亮。

时钟配置:
本例程采用HSI56MHz作为系统时钟，关于系统时钟如何配置，请参考对应用户手册或者
RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA系列

涉及外设:
COMP1 / 2
GPIO
涉及管脚:
LED1 / 2 / 3 / 4:
PC10, PC11, PC12, PD2
COMP:
PA1
注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估板，重启评估板;
2. 通过PA1引脚输入模拟电压。
