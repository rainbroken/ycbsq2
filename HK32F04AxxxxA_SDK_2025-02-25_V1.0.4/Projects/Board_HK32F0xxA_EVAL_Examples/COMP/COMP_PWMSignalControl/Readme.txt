******************************例程详细描述*****************************
功能描述:
本例程将COMP输出结果作为TIM1刹车信号的输入。

通过配置COMP1：
COMP1负端：Vrefint = 1.2V
COMP1正端：PA1

配置TIM1输出PWM信号，可以通过示波器观察PA8引脚PWM信号输出，当COMP1输出信号为
1时，触发TIM1刹车信号，TIM1停止输出PWM。

时钟配置:
本例程采用HSI56MHz作为系统时钟，关于系统时钟如何配置，请参考对应用户手册或者
RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA系列

涉及外设:
COMP1
TIM1
涉及管脚:
TIM1:
PA8
COMP1:
PA1
注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估板，重启评估板;
2. 通过PA1引脚输入模拟电压。
