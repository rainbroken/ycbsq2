******************************例程详细描述*****************************
功能描述:
这个例子展示了如何:
―配置以HSI56为时钟源作为系统时钟源
―使用EXTI来控制使能 / 禁用低速内部(LSI) RC振荡器(约40khz)
―每次BUTTON_KEY按钮被按下，EXTI_Line4 - 15中断被生成，
ISR当LSI开关关闭时使能它，当它打开时使能它。

当单板上有4个led可用时当LSI关闭时，LED3翻转闪烁
当LSI打开时，LED4翻转闪烁

LSI时钟波形在MCO1引脚PA8上可用，并且可以捕获在示波器上。

时钟配置:
本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA

涉及外设:
RCC
GPIO
涉及管脚:
MCO（PA8）
BUTTON_KEY（PB8）
注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。
如何使用:
1. 直接编译程序下载到评估版，重启评估版;
2.通过对PB8按键操作, 开启关闭LSI, 通过使用示波器查看PA8输出
