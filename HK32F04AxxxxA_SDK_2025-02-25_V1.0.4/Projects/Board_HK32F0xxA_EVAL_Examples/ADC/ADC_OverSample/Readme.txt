******************************例程详细描述*****************************
功能描述:
本例程描述了ADC采样的内部触发配置使用。

通过调用库函数完成ADC的采样和触发配置，配置过采样功能，选择256倍
过采样，过采样移位8位，通过通道10采样电压。

每次采样完成后计算结果由串口发送数据到终端。

时钟配置:
本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，需要在
Uart_Config函数之前完成，关于系统时钟如何配置，请参考对应用户手册或者
RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA系列

涉及外设:
USART1
ADC
涉及管脚:
USART1:
PA9 PA10
ADC:
PC0

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估板，重启评估板;
2. 通过串口打印判断ADC转换数据是否正确;
