******************************例程详细描述*****************************
功能描述:
1. 在TIM2输入引脚中接收到外部信号的上升沿后，使用TIM外围设备生成单脉冲模式；
2. TIM2CLK = 56MHz，TIM2计时器时钟设置为28MHz；
3. 重装载值为65535，因此最大的触发输入频率为28000000 / 65536 = 427.2 Hz；
4. TIM_Pulse为延迟时间，delay =  CCR1 / TIM2 counter clock = 585.1 us；
5. (TIM_Period - TIM_Pulse)定义的是一个脉冲的值，One Pulse value = (TIM_Period - TIM_Pulse) / TIM2 counter clock = 1.75 ms；
6. 如果测试，建议channel 2 输入信号使用频率计输入，如果使用杜邦线输入，会产生GPIO误触，影响观察结果。
时钟配置:
1. 本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
2. 关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA

涉及外设:
TIM2

涉及管脚:
PA1 - TIM2 Channel 2
PA5 - TIM2 Channel 1

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估版，重启评估版;
