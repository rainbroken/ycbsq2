******************************例程详细描述*****************************
功能描述:
1. 在PWM（脉宽调制）模式下配置TIM3；
2. 系统时钟为56MHz，TIM3CLK = 56MHz，TIM3计数器时钟频率为42KHz；
3. CH1频率为42KHz，占空比为50 % ，CH2频率为42KHz，占空比为37.5 % ，
CH3频率为42KHz，占空比为25 % ，CH4频率为42KHz，占空比为12.5 % 。
时钟配置:
1. 本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
2. 关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA

涉及外设:
TIM3

涉及管脚:
PA6 - TIM3 CH1
PA7 - TIM3 CH2
PB0 - TIM3 CH3
PB1 - TIM3 CH4

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估版，重启评估版;
