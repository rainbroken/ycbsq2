******************************例程详细描述*****************************
功能描述:
1. 在级联模式下同步TIM外围设备
2. 配置TIM1作为主计时器，ETR高电平触发，PWM输出功能被使能，TIM1更新事件作为触发输出；
3. 配置TIM2作为TIM1的从模式，作为TIM3的主模式，PWM功能被使能，ITR0（TIM1）作为输入触发，
Gated mode被使用，因此启动和停止TIM2计时器通过TIM1的触发输出事件（TIM1的更新事件）；
4. 配置TIM3作为TIM2的从模式，PWM功能被使能，ITR1（TIM2）作为输入触发，Gated mode被使用，
因此启动和停止TIM3计时器通过TIM2的触发输出事件（TIM2的更新事件）；
5. 系统时钟为56MHz，TIM1运行在频率为 56MHz APB总线上；
6. TIM1输出的PWM波形频率为218.75KHz，TIM2输出的PWM波形频率为54.688KHz，
TIM3输出的PWM波形频率为13.672KHz，占空比都为25 % 。
时钟配置:
1. 本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
2. 关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA

涉及外设:
TIM1
TIM2
TIM3

涉及管脚:
PA8 - TIM1 CH1
PA5 - TIM2 CH1
PA6 - TIM3 CH1

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估版，重启评估版;
