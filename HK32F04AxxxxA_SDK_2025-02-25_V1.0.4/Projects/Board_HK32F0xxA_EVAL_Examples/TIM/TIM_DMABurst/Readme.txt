******************************例程详细描述*****************************
功能描述:
1. 使用TIM1 DMA突发特性更新TIM1通道1周期和占空比；
2. DMA使能之前，TIM1输出PWM波形的频率为427.24Hz，占空比为93.75 % ；
3. 每次更新DMA请求，DMA将从ARR寄存器开始向定时器寄存器进行3次半字传输。
4. 在DMA更新请求时，0x0FFF将被传输到ARR，0x0000将被传输至RCR，0x0555将被传输给CCR1；
5. 最终TIM3的频率变为6.83KHz，占空比为33.33 % ；
时钟配置:
1. 本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
2. 关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA

涉及外设:
TIM1

涉及管脚:
PA8 - TIM1 Channel 1

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估版，重启评估版;
