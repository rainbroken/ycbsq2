******************************例程详细描述*****************************
功能描述:
1. 此示例提供了一个小型应用程序，使用SPI的poll方式处理SPI通信数据；
2. 此应用需要使用两块HK32应用板（A和B板），A板作为master，B板作为slave，连线方式如下：
*------------------------------------------------------------------------------*
|                BOARD A                                BOARD B                |
|         ____________________                   ____________________          |
|        |                    |                 |                    |         |
|        |                    |                 |                    |         |
|        |     __________     |                 |     __________     |         |
|        |    | SPI Master | ____ | _______SCK_______ | ____ | SPI Slave |    |         |
|        |    |  Device  | ____ | _______MISO______ | ____ |  Device  |    |         |
|        |    | __________ | ____ | _______MOSI______ | ____ | __________ |    |         |
|        |                    |                 |                    |         |
|        |  O LD1             |                 |  O LD1             |         |
|        |  O LD2    Joystick |                 |  O LD2    Joystick |         |
|        |  O LD3        _    |                 |  O LD3        _    |         |
|        |  O LD4       | _ |   |                 |  O LD4       | _ |   |         |
|        |                    |                 |                    |         |
|        |             GND O-- | ---------------- - | --O GND             |         |
|        | ____________________ |                 | ____________________ |         |
|                                                                              |
|                                                                              |
*------------------------------------------------------------------------------ *
3. 主机模式和从机模式下，SPI配置为全双工中断模式；
4. 主设备等待按键高电平的时候，触发SPI发送数据指令；
5. 从设备一直等待数据接收；
6. 最终通过接收到的数据和发送的数据进行对比，如果一致则表面发送接收成功，否则失败；
7. SysTick被配置为每10ms产生一次中断，以提供精准的计时。

时钟配置:
1. 本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
2. 关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA

涉及外设:
SPI
GPIO
SysTick

涉及管脚:
SPI_SCK - PA5
SPI_MISO - PB4
SPI_MOSI - PA7
SPI_CS - PA15
PA0 - BUTTON_SEL

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估版，重启评估版;
