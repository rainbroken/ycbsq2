******************************例程详细描述*****************************
功能描述:
本例程演示进入WFI模式并使用外部中断从该模式唤醒。

当用户按下按键PC13后，芯片进入WFI模式，功耗降低，再次按下
按键后，产生一个从WFI模式退出的中断，LED3用于指示MCU是否
处于WFI模式。
时钟配置:
本例程采用HSI56MHz作为系统时钟，关于系统时钟如何配置，请参考对应
用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA系列

涉及外设:
GPIO
EXTI
涉及管脚:
KEY:
PC13

LED3:
PC12


注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估板，重启评估板;
2. 按下PC13按键，系统进入WFI模式，LED3不再闪烁；
3. 再次按下PC13按键，系统退出WFI模式，LED3继续闪烁。
