******************************例程详细描述*****************************
功能描述:
本例程配置SysTick产生1ms的时钟基准。

本例程基于SysTick中断，实现了一个延时函数Delay.  4个LED通过Delay控制，进行翻转。

时钟配置:
本例程采用HSI56M，SysTick时钟由AHB时钟（HCLK）提供。
用户可以通过修改hk32f0xxa_conf.h中的SYSCLK_SOURCE修改时钟
细节请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32C005xx全系列

涉及外设:
USART1

涉及管脚:
LED1    PC10
LED2    PC11
LED3    PC12
LED4    PD2

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估版，重启评估版;
2. 通过LED闪灯或示波器测量确认SysTick是否正确;
