******************************例程详细描述*****************************
功能描述:
本示例旨在展示如何使用CRC外设进行计算消息的32位CRC校验和。
CRC外设固定使用32位CRC多项式：
X32 + X26 + X23 + X22 + X16 + X12 + X11 + X10 + X8 + X7 + X5 + X4 + X2 + X + 1
CRC初始化值为0。
消息被声明为一个数组，并标记为“CRCBuffer”。
一旦计算出CRC值，就将计算出的CRC值与预期值进行比较
在匹配的情况下，绿色led LD1打开。否则，led LD3打开。
并伴随相应的串口调试信息打印，再PC端串口调试助手查看

时钟配置:
本例程采用HSI56MHz作为系统时钟，用户如果需要更改系统时钟配置，
需要配置修改SYSCLK_SOURCE宏定义的。
关于系统时钟如何配置，请参考对应用户手册或者RCC库函数及RCC相关例程。

适用芯片:
HK32F04AxxA系列

涉及外设:
CRC

USART1
USART1配置如下:
―波特率 = 115200
―字长 = USART_WordLength_8b
―停止位 = USART_StopBits_1
―奇偶校验 = USART_Parity_No
―硬件流控制被禁用(RTS和CTS信号)
―使能接收和发送

涉及管脚:
USART1 TX - PA9
USART1 RX - PA10

注意事项:
1. 推荐使用MDK5.00及其以上版本编译本工程（MDK - ARM）
或者IAR8.22及其以上版本编译本工程(EWARM);
2. 使用之前检测电源VCC、GND等是否正常。

如何使用:
1. 直接编译程序下载到评估版，重启评估版;
